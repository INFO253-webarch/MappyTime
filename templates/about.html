About MappyTime

How to use the tool

MappyTime is an easy tool to visualize latitude/longitude points on a map! Using a csv file that contains multiple coordinates, you can easily plot many points to see how they relate relationally. For example, if you acquired data points of plant locations as well as insect locations and combined them into one file, you could easily see which species of plants and insects correlate! Customize your own data to build a beautiful map, better understand your data, and share it using the custom URL shortener !

What you'll need
